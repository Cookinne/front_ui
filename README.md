## 启动开发环境

### 安装依赖

``` bash
yarn
```

或者

``` bash
npm install
```

### 启动开发环境

``` bash
npm start

```

### 浏览器访问：[点这里](http://localhost:9901)

## 打包

``` bash
npm run build
```
